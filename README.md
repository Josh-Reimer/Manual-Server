Manual_Server is a simple web app built with python3 flask and pure javascript for managing all your pdf manuals.

INSTALLATION

pip install flask

git clone https://github.com/Josh-Reimer/Manual-Server --depth 1 --branch=main ~/manual-server

 cd manual-server
 
 python app.py
